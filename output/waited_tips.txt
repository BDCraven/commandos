You can stage all modified and deleted files without new files using 'git add -u'
Undo/revert a specific commit with 'git revert <commit id>'
You can stage all new and modified changes, except for deleted changes using 'git add .'
You can stage new, modified and deleted files using 'git add -A'
You can pick a particular branch to pull from using 'pull origin <branch_name>'
You can undo the changes in <filename> since the last commit using `git checkout <filename>'
